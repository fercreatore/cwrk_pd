Cómo funciona el Factor Estacional

El factor estacional ajusta las comisiones para que vender en meses difíciles (OI: Mar-Ago) se premie más que vender en diciembre cuando la venta fluye sola.

Fórmula: Comisión Final = Venta x % Comisión (según banda de margen) x Factor Estacional

Ejemplo: Un vendedor que vende $5M en marzo con 48% margen → Comisión = $5M x 4% x 1.3 = $260,000

El mismo vendedor vendiendo $5M en diciembre → Comisión = $5M x 4% x 0.8 = $160,000
