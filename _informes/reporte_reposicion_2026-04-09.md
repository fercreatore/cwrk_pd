## Metodologia

1. **Quiebre**: Reconstruccion de stock mes a mes hacia atras (12 meses). Meses con stock_inicio <= 0 = QUEBRADO.
2. **Velocidad real**: Ventas solo de meses NO quebrados / cantidad de meses NO quebrados.
3. **Estacionalidad**: Factor Abr-May vs promedio anual (3 anios historia 2023-2025), por proveedor.
4. **Vel ajustada** = vel_real x factor_estacional. Dias_stock = stock_actual / (vel_ajustada / 30).
5. **Pedir** = (vel_ajustada/30 x 60 dias) - stock_actual. Cobertura objetivo = 60 dias.
