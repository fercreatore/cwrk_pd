# Registro de puertos — cowork_pedidos

Todas las apps Streamlit del proyecto usan puertos fijos para evitar conflictos.
**Antes de agregar una app nueva, elegir el siguiente puerto libre de esta lista.**

| Puerto | App | Carpeta | Comando |
|--------|-----|---------|---------|
| 8501 | Carga de pedidos (principal) | `.` (raíz) | `streamlit run app_carga.py` |
| 8502 | App H4 | `.` (raíz) | `streamlit run app_h4.py` |
| 8503 | Family Office | `_family_office/` | `streamlit run _family_office/app_family_office.py` |
| 8504 | *(libre)* | | |
| 8505 | *(libre)* | | |
